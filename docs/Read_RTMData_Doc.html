Routine Read_RTMData

Documentation:
Controls the reading of the long- and short-wave RTM data files.

This routine is a straightforward calling routine for the separate long and shortwave RTM data files. For a definition of 'long' and 'short' wave in the context of atmospheric RTM data and other aspects of the ECP see the overview on indexing.
