ORAC

Documentation on the ORAC retrieval software. ORAC is an implementation of optimal estimation methodology to the inversion of imagery, for advanced cloud and aerosol products. This community code can be applied to any visible/near infrared/infrared instrument providing the correct inputs are prepared. The following web pages contain the documentation on the ORAC community retrieval software. The links below provide guidance on what input files are required, information on the output, how to download and install the code. If you have any questions or suggestions then please contact the people listed below.
