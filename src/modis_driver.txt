input decsription
-----------------
1) absolute path of top level directory where preprocesssing files are stored
2) filename body of preprocessing files without individual suffixes
3) absolute path to output directory
4) absolute path to sad directory
5)instrument
6)number of channels used(i.e. line 4): number in input file(i.e line 5) this is different to accomodate processing with 1.6 or 3.7 channel for example
7) which channels to use
8) index of channels in input file
9) phase to process

example usage
------------------
In order to drive ORAC the following text file should be set.

setenv ORAC_TEXTIN ~/orac_svn/v11/src/test_orac_textin.txt


previously this environment variable was required.
setenv ORAC_DRIVER ~/orac_svn/v6/trunk/drivers/modis/MODIS_WAT_200806201240_380-750_netcdf_new.dri

comments about channel numbering in case of avhrr
!3=3B,6=3A (that's how it is in the HDF5 input file)
channel_info%channel_ids_instr= (/ 1, 2, 3, 4, 5, 6 /)
!numbering wrt to increasing wl, starting at 1
!3=3A,4=3B
channel_info%channel_ids_abs= (/ 1, 2, 3, 4, 5, 6 /)
